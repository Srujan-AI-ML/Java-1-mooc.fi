# University of Helsinki Java Programming MOOC

Welcome to my Java development repository! This project tracks my daily code progression through the "MOOC.fi Java Programming" course by the University of Helsinki.

The goal of this repository is to build an strong foundation in Object-Oriented Programming (OOP) and software principles as I prepare for advanced computer science studies.
